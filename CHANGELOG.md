# Changelog

All notable changes to Ribbit Survival are documented here. The format follows
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and versions aim to
follow [Semantic Versioning](https://semver.org/). While the pack is in `0.x`,
it is **alpha**: content, balance, and the mod list can change between releases.

## [Unreleased]

## [0.2.0] - unreleased

### Changed
- **Productive Frogs bumped to `2.0.0-alpha.3`** ("Field Guide").

### Added
- **Modonomicon** (`2.1.1`) - the guide engine PF's new field guide runs on.
  Craft the guide from a book + slime ball for an illustrated, start-to-finish
  walkthrough of the mod: the frog loop, every species, the machines, the
  Terrarium, and the boss altars, each with a rotatable 3D build diagram. PF
  ships the book as data (inert on its own); Modonomicon is what renders it.

### Notes
- 79 mods, all CurseForge-sourced (clean manifest export, no bundled jars).

## [0.1.0] - 2026-07-18

First build of the pack. Uploaded to CurseForge as an **alpha** release (file `8458262`);
live once first-file moderation clears.

### Added
- **Productive Frogs** as the marquee content mod (CF project `1552728`), pinned
  to the `26.1.2` alpha - frogs that farm resources, hunt mobs for their drops,
  and arm boss altars, in a **normal survival** world (not skyblock).
- A lean **tech spine**: Applied Energistics 2 + Powah, with the featured hook of
  Powah RF powering Productive Frogs' Equivalence (EE) transmutation lane.
- **Apotheosis** RPG layer (Apotheosis, Apothic Attributes/Enchanting/Spawners).
- Worldgen: Terralith, Tectonic, YUNG's Better Mineshafts, Structory, Explorer's Compass.
- A curated performance + QoL layer swept from the sibling Sky Frogs pack and
  verified on 26.1.2 (Sodium, Lithium, FerriteCore, ModernFix, JEI, Jade, the
  Sophisticated storage suite, Xaero's maps, and more).
- Config override enabling the Equivalence (EE) lane (`[equivalence] enabled = true`)
  and asserting normal-world identity (`[spawnery] enabled = false`).
- Pack logo + CurseForge icon (`branding/`).
- **78 mods total** (all CurseForge-sourced for a clean CF manifest export).

### Notes
- Powah ships as an alpha build (`7.0.4-alpha`) - the only 26.1.2 build available,
  and the one this pack is built and tested against.
